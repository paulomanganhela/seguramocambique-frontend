
SeguraMoçambique - Frontend Completo

Instruções:
1. Substitua todos arquivos no seu projeto seguramocambique-frontend
2. Depois, no terminal:
    git add .
    git commit -m "Atualização completa da Home Page SeguraMoçambique"
    git push origin main
3. O Vercel atualizará automaticamente o site
4. Home: https://seguramocambique-frontend.vercel.app/
5. Administração: https://seguramocambique-frontend.vercel.app/admin.html

Após configurar, personalize imagens dentro da pasta assets/.
